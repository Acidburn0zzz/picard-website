MusicBrainz Picard

Picard is a cross-platform music tagger written in Python.

Download Picard v1.2 for Windows Download Picard v1.2 for OS X 10.7+ Download Picard v1.2 for Linux Distributions

Multiple formats

Picard supports all popular music formats like mp3, flac, ogg, m4a, wma, wav and a lot of others.

AcoustID

It is capable of using Audio fingerprints to identify your songs.

CD Lookups

Picard can lookup entire music CDs with a click.

Opensource

Picard is licensed under GPL 2.0 or later and is hosted on Github where it is actively developed by some awesome developers.

Plugin Support

If you need a particular feature, you can choose from hundreds of available plugins or can write your own ones in Python.
