MusicBrainz Picard

Picard is a cross-platform music tagger written in Python.

Multiple formats

Picard supports all popular music formats like mp3, flac, ogg, m4a, wma, wav and a lot of others.

AcoustID

It uses AcoustID audio fingerprints, allowing a song to be identified by the actual music, so it can be matched even if you have no metadata.

CD Lookups

Picard can lookup entire music CDs with a click.

Opensource

Picard is licensed under GPL 2.0 or later and is hosted on Github where it is actively developed by some awesome developers.
